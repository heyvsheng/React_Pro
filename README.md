#### 全局状态管理
是指多个页面需要共享或者跟踪变化的变量，可以放到全局来统一维护，而不是每个页面分别维护和获取。
适合作为全局状态的数据：已登录用户信息
在Vue中，主流的状态管理库有Vuex和Pinia；在React中，主流的状态管理是Redux

#### router.push()和router.replace()有什么区别？
push是在浏览器里新加一个页面，如果按返回按钮就是返回‘上’一个页面
replace是替换掉页面，如果按返回按钮就是返回‘前’一个页面



#### menu.tsx的疑惑
路由：决定了输入什么URL（eg：/user/login）显示什么组件。它是整个网站的地图
菜单：是侧边栏或导航栏上显示的按钮
逻辑：我们通常不希望用户在登录之后，侧边栏里还一直显示着一个“登录”或“注册”的按钮。所以，登录和注册页面虽然有路由，但它们不属于导航菜单。
